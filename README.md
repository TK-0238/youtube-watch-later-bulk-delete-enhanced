# 🗑️ YouTube 後で見る 一括削除 - 強化版

YouTubeの「後で見る」プレイリストから動画を効率的に一括削除できるChrome拡張機能です。

## 🎯 こんな問題を解決します

- ✅ **「後で見る」に動画が溜まりすぎて整理が大変**
- ✅ **一つずつ削除するのが面倒**
- ✅ **日本語のタイトルで動画を検索したい**
- ✅ **条件を絞って選択的に削除したい**

## ✨ 主な機能

### 🔍 **強化された検索機能**
- 日本語（ひらがな・カタカナ・漢字）の自由な検索
- 英数字の全角・半角を自動で判別
- 部分一致検索で目的の動画を素早く発見

### ☑️ **柔軟な選択機能**
- 個別選択：好きな動画だけをピンポイント選択
- 全選択：表示中の動画をワンクリックで選択
- フィルター連携：検索結果から選択して削除

### 🚀 **安全な一括削除**
- 削除前に確認ダイアログを表示
- 削除進捗をリアルタイム表示
- 削除後に選択状態を自動リセット

## 📱 対応ブラウザ

- ✅ Google Chrome (推奨)
- ✅ Brave Browser
- ✅ Microsoft Edge
- ⚠️ その他のChromiumベースブラウザ（要テスト）

## 🚀 インストール方法

### ステップ 1: ファイルをダウンロード

1. **このリポジトリを丸ごとダウンロード**
   - 画面上部の緑色「Code」ボタン → 「Download ZIP」をクリック
   - ダウンロードしたZIPファイルを解凍

### ステップ 2: アイコンファイルを準備

**重要：** 拡張機能が正常に動作するにはアイコンファイルが必要です。

1. **`icons/` フォルダ内の `README.md` を確認**
2. **以下のアイコンを作成（内容は何でもOK）**：
   - `icon16.png` (16×16ピクセル)
   - `icon48.png` (48×48ピクセル) 
   - `icon128.png` (128×128ピクセル)
3. **作成したファイルを `icons/` フォルダに配置**

### ステップ 3: ブラウザに拡張機能をインストール

1. **拡張機能管理ページを開く**
   - Chrome: `chrome://extensions/`
   - Brave: `brave://extensions/`
   - Edge: `edge://extensions/`

2. **デベロッパーモードを有効化**
   - 右上の「デベロッパーモード」トグルをONにする

3. **拡張機能を読み込む**
   - 「パッケージ化されていない拡張機能を読み込む」ボタンをクリック
   - 解凍したフォルダを選択
   - 「フォルダーの選択」をクリック

### ステップ 4: インストール確認

✅ **成功の目印**:
- 拡張機能一覧に「YouTube 後で見る 一括削除」が表示される
- ブラウザのツールバーにアイコンが表示される
- 拡張機能が「有効」状態になっている

## 📖 使い方

### 基本操作

1. **YouTubeの「後で見る」ページにアクセス**
   ```
   https://www.youtube.com/playlist?list=WL
   ```

2. **拡張機能のUIを確認**
   - ページ上部に「🗑️ YouTube後で見る 一括削除」が表示されます

3. **一括削除モードを有効化**
   - 「一括削除モードを有効化」ボタンをクリック
   - 各動画の右上角にチェックボックスが表示されます

### 動画の選択方法

#### 👆 個別選択
- 削除したい動画のチェックボックスをクリック
- 選択数がリアルタイムで表示されます

#### 🔍 検索して選択
1. **検索窓にキーワードを入力**（例：「React」「講座」「プログラミング」）
2. **該当する動画のみが表示されます**
3. **「✅ すべて選択」で表示中の動画を一括選択**

#### ⚡ 全選択・全解除
- **「✅ すべて選択」**: 表示中の動画をすべて選択
- **「❌ 選択解除」**: 選択をすべて解除

### 削除の実行

1. **動画を選択後、削除ボタンをクリック**
   - 「選択した○個の動画を削除」ボタンが有効になります

2. **確認ダイアログで「OK」をクリック**
   - 削除は元に戻せませんのでご注意ください

3. **削除進捗を確認**
   - プログレスバーで進捗状況が表示されます
   - 完了後、成功通知が表示されます

## 🔧 よくある問題と解決方法

### ❓ 拡張機能のUIが表示されない

**解決方法**:
1. ページを完全にリロード（`Ctrl + Shift + R`）
2. 正確に「後で見る」ページにいることを確認
3. 他のYouTube関連拡張機能を一時的に無効化

### ❓ チェックボックスが表示されない

**解決方法**:
1. 「一括削除モード」をオフ・オンしてみる
2. ページをリロード
3. 動画が実際に存在することを確認

### ❓ 削除が動作しない

**解決方法**:
1. YouTubeにログインしていることを確認
2. 手動で動画の「・・・」メニューから削除できるか確認
3. ページをリロードしてから再試行

### ❓ 「読み込み中」のまま止まる

**解決方法**:
1. ブラウザの拡張機能ページで「🔄 更新」をクリック
2. ブラウザを再起動
3. 拡張機能を一度無効化→有効化
